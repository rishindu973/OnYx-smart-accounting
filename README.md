# 🧾 OnYx: Smart Accounting & Governance

OnYx is an AI-driven financial governance platform designed to streamline accounting workflows, enforce auditability, and deliver intelligent financial insights. It combines a modern full-stack architecture with scalable backend systems and a rich, interactive frontend.

---

## 🚀 Tech Stack

* **Frontend:** Next.js 14, Tailwind CSS, Framer Motion, Recharts
* **Backend:** Prisma ORM, PostgreSQL
* **Architecture:** Full-stack (App Router + Server Actions)

---

## 💡 Key Features

* AI-powered transaction classification and smart suggestions
* Audit-safe accounting with a strict no-deletion policy
* Real-time financial dashboards and analytics
* Scalable and modular full-stack architecture

---

## 📁 Project Structure

```bash
src/
├── app/               # Routing & core application pages
├── components/        # Reusable UI components
├── contexts/          # Global state management
├── hooks/             # Custom React hooks
├── lib/               # Backend logic & computation engines
├── services/          # AI logic & processing
└── types/             # Shared TypeScript types
```

---

## 🏗️ Architecture Overview

* **Frontend:** Built using Next.js App Router with a focus on performance and user experience
* **Backend Logic:** Implemented via Server Actions with Prisma ORM
* **Database:** PostgreSQL ensuring reliability and scalability
* **AI Layer:** Handles intelligent suggestions, vendor mapping, and automation

---

## 📊 Core Modules

* **AI Powered Document Processing**
  Upload receipts, invoices, or bills with instant data extraction and confidence scoring

* **Real Time Spending Controls**
  Enforce daily spending limits through a governance calendar

* **Audit Ready System**
  Permanent transaction logging with secure reversal-based corrections

* **Intelligent Review Interface**
  Side-by-side document validation with smart vendor detection and anomaly flags

* **Automated Bank Reconciliation**
  Automatic matching of ledgers and bank statements with discrepancy detection
